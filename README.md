A **HakySoft Labs** é uma plataforma web estática avançada inspirada em estações de trabalho de segurança ofensiva (SOC / Kali Linux). Foi desenvolvida para centralizar o acesso a laboratórios de treino reais (VulnHub) e simular um ecossistema tático de hacking diretamente no navegador.

---

## ⚡ Funcionalidades Principais

* **Bypass Auth Gate:** Ecrã de login encriptado com proteção visual estilo *glitch matrix*.
* **Repositório VulnHub Integrado:** Links diretos e espelhos estáveis (via Internet Archive) para descarregar máquinas virtuais vulneráveis clássicas (`Mr-Robot: 1`, `Kioptrix: Level 1`, `DC-1`, `SickOs: 1.2`).
* **Consola Interativa Kali Linux:** Terminal virtualizado com comportamento real (scroll dinâmico travado) que aceita comandos como:
  * `help` (Lista de comandos)
  * `sudo su` (Elevação de privilégios para Root)
  * `nmap`, `gobuster`, `msfconsole`, `hydra`, `ping` e `whoami`.
* **Monitor de Rede Promíscuo (eth0):** Painel tático que simula a captura de pacotes, tráfego inbound/outbound e monitorização de conexões ativas a cada segundo.
* **Arsenal Técnico de Elite:** Acesso rápido a ferramentas fundamentais da indústria como Exploit-DB, CyberChef, SecLists e geradores de Reverse Shells.
* **Módulos de Tempo:** Relógio operacional sincronizado e contador de uptime da sessão ativa.
