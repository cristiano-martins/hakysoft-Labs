A **HakySoft Labs** é uma plataforma web estática avançada inspirada em estações de trabalho de segurança ofensiva (SOC / Kali Linux). Desenvolvida por **Cristiano Martins**, foi desenhada para centralizar o acesso a laboratórios de treino reais (VulnHub) e simular um ecossistema tático de hacking diretamente no teu navegador.

---

## ⚡ Funcionalidades Principais

* **Bypass Auth Gate:** Ecrã de login encriptado com proteção visual estilo *glitch matrix*.
* **Repositório VulnHub Integrado:** Links diretos e espelhos estáveis (via Internet Archive) para descarregar máquinas virtuais vulneráveis clássicas (`Mr-Robot: 1`, `Kioptrix: Level 1`, `DC-1`, `SickOs: 1.2`) sem erros de *Not Found*.
* **Consola Interativa Kali Linux:** Terminal virtualizado com comportamento real e altura rígida (scroll dinâmico travado) que aceita comandos táticos como:
  * `help` (Lista o arsenal de comandos)
  * `sudo su` (Elevação de privilégios para Root com alteração visual de prompt)
  * `nmap`, `gobuster`, `msfconsole`, `hydra`, `ping` e `whoami`.
* **Monitor de Rede Promíscuo (eth0):** Painel que simula a captura de pacotes ativos, fluxo de tráfego de dados e monitorização de conexões em tempo real.
* **Arsenal Técnico de Elite:** Acesso rápido integrado para ferramentas reais como Exploit-DB, CyberChef, SecLists e geradores de Reverse Shells.
* **Controlo de Tempo:** Relógio operacional sincronizado em tempo real e contador de uptime da sessão ativa do operador.
